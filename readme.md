# @capire/s4

This sample showcases how to provide pre-built integration packages for SAP S/4HANA integration.

> [!CAUTION]
> This is just a sample, not intended for production use. It is meant to demonstrate how to create a CAP plugin and how to use it in your application. It is not optimized for performance or security.


### Plug & Play

Following the plug & play principle of CAP plugins, adding this dependency to your project will automatically register a service binding endpoint for S4BusinessPartnerService.
